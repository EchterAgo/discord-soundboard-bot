Help & Tips
Play Modes
  • Instant: Interrupts current playback and plays immediately
  • Queue: Adds to the end of your personal queue
  • Play Next: Plays after the current sound finishes

Click the mode button in the header to cycle through modes.

Keyboard Shortcuts
  • Ctrl + Click: Play in instant mode (override current mode)
  • Shift + Click: Play in queue mode (override current mode)
  • Ctrl + Shift + Click: Play in next mode (override current mode)
  • Ctrl + F: Focus search box
  • Tab / Shift+Tab: Navigate between elements
  • Arrow Keys: Navigate sound buttons in grid
  • Enter / Space: Activate focused button
  • Escape: Close modals and dropdowns
  • ?: Toggle this help panel
  • S: Toggle settings panel
  • Q: Toggle queue status panel
  • F: Toggle fullscreen mode
  • 1/2/3: Switch to Custom/Recent/All view

Hold modifiers to see the mode indicator change.

Search Behavior
  • Search shows matching sounds from your collection
  • Recently played sounds appear first in results
  • Use ↑ ↓ arrow keys to navigate
  • Press Enter to play selected sound
  • Search is case-insensitive and matches partial names
Custom Buttons
  • Right-click a button to edit it
  • Right-click sounds in "All Sounds" to add as button
  • Drag and drop to rearrange buttons
  • Choose from 7 different colors for organization
  • Export/import your button configuration
Recent Sounds

Your last 20 played sounds are automatically saved and appear in the "Recent Sounds" view. Recent sounds are prioritized in search results for quick access.

Queue Management
  • View active streams and queued sounds at the bottom
  • Each user has their own queue
  • Click queue headers to expand/collapse
  • Remove individual items with the button
  • Use the Stop button to clear playback and queues
